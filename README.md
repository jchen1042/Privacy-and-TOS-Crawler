# TOS Privacy Policy Crawler

A comprehensive web application that crawls, analyzes, and provides insights on Terms of Service and Privacy Policy documents from websites using AI-powered text mining techniques.

## Project Overview

This application allows users to:
**Crawl** Terms of Service and Privacy Policy documents from websites
**Analyze** them using AI-powered text mining
**Extract** key insights including summaries, word frequencies, and risk indicators
**Track** analysis history and favorite documents

## Core Features

### Web Crawling
- Target documents: Terms of Service, Privacy Policy, Terms and Conditions, Terms of Use
- Focus domains: .com (retail, technology, health), .edu, .org
- Smart caching strategy (30-day cache for analyzed documents)

### Text Mining Analysis
- **Summaries**: 100-word summary and one-sentence summary
- **Word Frequency**: Top words with statistics
- **Text Mining Measurements**:
  - Word count
  - Sentence count
  - Average words per sentence
  - Flesch Reading Ease Score
  - Sentiment Analysis Score
   (a few more are under development, will be updated by the Beta)

### User Management
**Features**:
  - Search history tracking
  - Document favorites
  - Report downloads (planning to implement)
